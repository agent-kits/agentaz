# Contributing to AgentKits & AgentAz™

Thanks for your interest. AgentKits is an open registry of AI agent blueprints
built on the AgentAz™ governance specification.

## Ways to contribute

- **Improve a blueprint** — sharper failure modes, evaluation guidance, or
  governance detail. Edits flow from `scripts/kits-flagship.mjs`.
- **Propose an AgentAz field** — open an issue describing the gap. v1.0 is frozen
  to additive, backward-compatible changes; breaking proposals are tracked for 2.0.
- **Fix bugs / docs** — small, targeted PRs are welcome.

## Ground rules

- Keep changes minimal and targeted; preserve SEO and design.
- Run `npm run build` — it validates all kits and must stay green.
- By submitting a contribution you agree it is licensed under Apache-2.0 (code)
  and CC-BY-4.0 (specification text), per `LICENSE` and `LICENSE-CONTENT`.

## License of contributions
